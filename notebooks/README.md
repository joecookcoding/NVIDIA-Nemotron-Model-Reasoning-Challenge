# Notebooks

- `nvidia-nemotron-submission-demo.ipynb`: Preferred Kaggle submission notebook. Minimal offline LoRA path that installs from the offline wheelhouse, trains a small adapter, and packages `submission.zip`.
- `nemotron_leaderboard_engine.ipynb`: Kaggle competition analysis notebook for model attachment, GPU smoke testing, model loading, heuristic baselines, and submission scaffold discovery.
- `nvidia-nemotron-sfttrainer-training.ipynb`: Extended offline LoRA training notebook with a larger training/validation flow than the submission demo.
